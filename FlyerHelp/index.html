How to play Flyer

Your task is simple: shoot any object before it hits you.

This is your ship:

You control your ship with four commands, which by default are linked to the following keys:

Spin counter-clockwise
Spin clockwise
Thrust
Fire

You can also pause the game by pressing a fifth key, which by default is the space bar.

Hints

  • There is a limit to how many missles can be active at one time. Use them sparingly.
  • When you first appear you are in a protected state until you move. You might want to wait a bit.
  • Each type of object responds to your movements in a predictable way. Learn and adapt.
  • If your are moving too fast, bad things could happen. Don't be reckless.
